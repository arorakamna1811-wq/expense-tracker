# Expense Tracker (Python CLI Project)

This project is a simple expense tracker built using Python. It stores expense data in a file and allows users to manage their expenses through a command-line interface.

## 🔥 Features

- Add new expenses
- View all expenses
- Calculate total spending
- Data stored using JSON file

## 🛠️ Technologies Used

- Python
- JSON (for file storage)

## 🚀 How to Run

1. Make sure Python is installed
2. Download or clone the repository
3. Open terminal or command prompt
4. Run:

   python expense_tracker.py

## 📌 Concepts Used

- File handling
- JSON data storage
- Functions
- Loops and conditions
